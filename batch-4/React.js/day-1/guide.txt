1. create a folder 
2. create 'public' & 'src' folder inside it 
3. create index.html inside public folder , App.js | index.js | 
   App.css inside src folder.
4. open that folder in terminal (cd <foldername>)
5. in terminal do "npm init -y"
6. do "npm install react react-dom react-scripts"
7. inside package.json find "scripts" object & write ["start": "react-scripts start"]
8. on cmd/terminal do "npm start"  
9. create .gitignore file inside the main folder and write "/node_modules", in order to avoid pushing the node_modules in github
